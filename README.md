# Test klouzavého průměru

Byl použit výňatek z historických dat z cyklosčítačů v Praze
http://opendata.praha.eu/dataset/cykloscitace

## Requirementy

* python3.6
* python3-pip
* python3-distutils
* python3-venv
* python3-tk

## Spuštění

```sh
python3 -m venv ./venv
source ./venv/bin/activate
pip install -r requirements.txt
python3 main.py
```
